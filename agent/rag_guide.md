# RAG 시스템 가이드

## 개요

이 문서는 RAG (Retrieval-Augmented Generation) 시스템에 대한 설명서입니다.

## 주요 기능

- 문서 업로드 및 벡터화
- 자연어 질의응답
- 실시간 스트리밍 응답
- 다국어 지원 (한국어, 영어)

## 사용법

1. 문서를 업로드합니다
2. 질문을 입력합니다
3. AI가 문서를 참조하여 답변을 제공합니다

## 지원 파일 형식

- PDF 파일 (이미지 OCR 포함)
- 텍스트 파일 (.txt)
- 마크다운 파일 (.md)

이 시스템은 최신 AI 기술을 활용하여 정확하고 유용한 정보를 제공합니다.
